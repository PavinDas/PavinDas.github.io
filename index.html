> WiFried
WiFi Attacking Tool

Python-based tool to scan Wi-Fi networks, perform deauthentication attacks, capture WPA handshakes, and attempt password cracking.

  • List available Wi-Fi interfaces
  • Enable monitor mode & scan networks
  • Perform deauth attacks
  • Capture WPA handshakes
  • Crack passwords using aircrack-ng

> TrafficTrace
Network Traffic Monitor

Python-based tool for network traffic analysis, intrusion detection, and log monitoring.

  • Identifies active hosts using ARP
  • Parses network logs for anomalies
  • Shows top active IPs & protocols
  • Detects port scans & exfiltration

> CoreWatch
Linux System Monitor Tool

Bash script for system health checks, CPU/memory monitoring, and automated backups with rsync.

  • Monitors CPU & memory usage
  • Logs system health details
  • Top 5 memory-heavy processes
  • Checks for system updates
  • Backups using rsync

> TamperTracker
File Integrity Monitor

Python-based FIM that detects unauthorized file changes and creates Jira incidents automatically.

  • SHA-256 hashing for file tracking
  • Real-time tamper detection
  • Auto Jira ticket creation
  • Configurable monitoring intervals


Code
> NopeWare
Virus Scanner Tool

CLI utility to upload files to VirusTotal, fetch results, and highlight malicious detections.

  • Uploads files for scanning
  • Shows name, size & SHA-256
  • Displays AV verdicts
  • Flags malicious detections

Subdomain Finder Tool

Python script using a custom wordlist to discover hidden subdomains and paths.

  • Custom wordlist-based discovery
